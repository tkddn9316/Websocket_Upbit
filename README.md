# Websocket_Upbit
업비트 API와 WebSocket를 이용한 거래소 리스트 화면 입니다.

해당 앱은 거래소 리스트(Main)으로 이루어져 있습니다.
(추후 호가창 계획 있음)

|:---:|
|거래소 화면|

업비트에서 ticker(실시간 채결 내역)을 30개 불러와 앱에 세팅합니다.

초기 세팅 이후 곧바로 웹소켓이 작동하여 리스트를 갱신하게 됩니다.
